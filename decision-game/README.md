# Decision Game

A beginner-level text-based decision game built in Python.

## What I practiced
- Conditional logic
- Loops
- Functions
- State management (health system)
- User input handling

## Features
- Multiple choices (left/right)
- Health system using ❤️
- Second chance logic in cave
- Multiple endings
- Emoji-enhanced console UI

## How to run
```bash
python3 decision_game.py
